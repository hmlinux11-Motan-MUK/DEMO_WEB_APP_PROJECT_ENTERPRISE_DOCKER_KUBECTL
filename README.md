# 🚀 DEMO WEB APP PROJECT - ENTERPRISE (Docker + Kubernetes)

## 📌 Project Overview
This project demonstrates an enterprise-level web application deployment using Docker and Kubernetes.

It includes:
- Containerized application using Docker
- Kubernetes deployment (pods, services, scaling)
- MongoDB database integration
- Backend and frontend architecture

---

## 🏗️ Architecture
- Frontend: React / Web UI
- Backend: Node.js / Express
- Database: MongoDB
- Containerization: Docker
- Orchestration: Kubernetes (K8s)

---

## ⚙️ Features
- Scalable microservices architecture
- Container-based deployment
- Kubernetes auto-scaling
- Load balancing
- Database integration
- Easy deployment and management

---

## 🛠️ Technologies Used

## 🚀 Setup & Installation

### 1. Clone Repository
```bash
git clone https://github.com/hmlinux11-Motan-MUK/DEMO_WEB_APP_PROJECT_ENTERPRISE_DOCKER_KUBECTL.git
cd DEMO_WEB_APP_PROJECT_ENTERPRISE_DOCKER_KUBECTL

2. Run with Docker
docker-compose up --build

3. Deploy to Kubernetes
kubectl apply -f kubernetes/

4. Check Pods
kubectl get pods

🌐 Access Application
Frontend: http://localhost:3000
Backend: http://localhost:5000
